# Handshake MVP Product Requirements

## Goal
Help travel baseball coaches lock non-tournament games without text chaos.

## North Star Metric
Games Locked.

## MVP User Stories
- Coach creates a team profile.
- Coach adds tournament blackout dates.
- Coach searches verified teams by age and location.
- Coach sends a Handshake request.
- Receiving coach accepts, counters, or declines.
- Confirmed game is created only after both sides agree.
- System blocks conflicts caused by blackouts or existing confirmed games.
