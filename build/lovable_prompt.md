# Lovable/Bolt Prompt

Build a working web app called Handshake.

It is a light-mode SaaS dashboard for travel baseball coaches.

Core value:
Find opponents, manage blackout dates, propose games, counter, confirm, and lock games without text-message chaos.

Pages:
1. Login / Sign up
2. Dashboard
3. Team Profile
4. Team Directory
5. Team Detail
6. Blackout Manager
7. Propose Game
8. Request Detail
9. Confirmed Games

Workflow:
1. Coach creates team.
2. Coach searches teams by age group and location.
3. Coach sends a game request.
4. Receiving coach accepts, counters, or declines.
5. If accepted, create confirmed game.
6. Prevent confirmation if either team has a blackout or confirmed game on that date.

Primary CTA language:
- Send Handshake
- Counter
- Lock Game
- Confirm Game
